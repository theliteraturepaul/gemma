# Benglish/Hinglish Chat Normalizer

Kolkata chat users routinely write Bengali and Hindi in Roman script, mixed with English, within the same message. This breaks accessibility for elderly relatives, search/indexing tools, and anyone unfamiliar with the informal romanization conventions. This app detects the languages and produces a fully normalized version in native script, along with a clean English translation and a language-mix ratio.

**Track:** Local Language & Inclusion
